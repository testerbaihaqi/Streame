🔧 Panduan Setup Firebase (Baca Sebelum Mulai)

1
Buat Proyek Firebase — Buka console.firebase.google.com, klik "Add Project", beri nama, lalu klik Continue.
2
Daftarkan Web App — Di dashboard, klik ikon </> (Web), beri nama app, centang "Firebase Hosting", lalu klik Register. Salin kode firebaseConfig.
3
Tempel Config — Buka file index.html dan admin.html, cari bagian firebaseConfig di bagian atas, ganti semua nilai YOUR_... dengan nilai dari Firebase Anda.
4
Aktifkan Firestore — Di Firebase Console, pilih "Firestore Database" → "Create Database" → pilih mode "Test mode" (sementara) → pilih region asia-southeast2 (Jakarta).
5
Aktifkan Storage — Pilih "Storage" → "Get Started" → pilih mode "Test mode" → pilih region yang sama.
6
Buat Akun Admin — Pilih "Authentication" → "Get Started" → pilih "Email/Password" → Enable. Lalu di tab "Users", klik "Add User", masukkan email & password admin Anda.
7
Upload ke Firebase Hosting — Install npm install -g firebase-tools → login dengan firebase login → firebase init hosting → firebase deploy. Website langsung online dan bisa diakses dari HP!
8
Tambah Video — Klik "+ Tambah Video", isi judul, kategori, tahun, deskripsi. Untuk video: masukkan link YouTube/Google Drive di "Embed URL", atau upload file MP4 langsung. Klik Simpan → video langsung muncul di semua orang real-time!
💡
Tips Format Video — YouTube: paste URL biasa seperti https://youtube.com/watch?v=xxx — otomatis dikonversi. Google Drive: upload video ke Drive, klik kanan → "Get link" → ubah ke "Anyone with link", paste URL. Upload langsung: pilih file MP4 dari HP/PC.
